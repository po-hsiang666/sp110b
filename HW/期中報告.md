## 甚麼是作業系統OS (Operating System)?

從兩個角度出發
## OS服務的對象有二
```
*User(人)
*Program
```
## 使用者觀點
```
*OS 電腦使用者(USER)與電腦硬碟(Hardware)之間的介面，使得User易於使用Hardware

*提供一個讓 【User Program易於執行】的環境
```

## 系統觀點
```
*是一個資源分配者(Resource Allocator)
。解決資源使用上之衝突
。讓資源公平且有效的被利用
 -HW:CPU,Memory,I/O Device
 -SW:Share Files
 ```
