# 系統軟體
## 何謂系統程式
系統軟體                       | 應用軟體
-----------------------------|--------------------------------------------
簡介               | 開發工具 / Windows / Linux
gcc 編譯工具       | gcc / g++ / make / gdb
編譯器設計     | 運算式 / Lexer / 中間碼
組合語言             | 組合語言 / 處理器 / 虛擬機
目的檔             | ELF / 連結 / 載入
軟體工程              | 速度 / 正確性 / 測試
函式庫             | msys2 / database
Linux 作業系統       | Thread / 檔案 / 網路 / pipe / mmap
RISC-V 處理器      | 組合語言 / mini-riscv-os / xv6
